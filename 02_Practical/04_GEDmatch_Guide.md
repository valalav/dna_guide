# GEDmatch для начинающих

*Полное руководство по поиску родственников*

---

## Введение

GEDmatch — это бесплатная платформа для сравнения аутосомной ДНК. Она объединяет данные из разных тестовых компаний (23andMe, AncestryDNA, MyHeritage, FTDNA и других), позволяя найти родственников, которые тестировались в других лабораториях.

**Для участников AADNA:** GEDmatch — основной инструмент поиска живых родственников, особенно в диаспоре.

---

## Часть 1: Регистрация

### Шаг 1: Зайдите на сайт

Откройте [gedmatch.com](https://www.gedmatch.com)

Вы увидите главную страницу с двумя колонками:
- Левая — для зарегистрированных пользователей
- Правая — регистрация и информация

### Шаг 2: Создайте аккаунт

1. Нажмите **"Register"** (Зарегистрироваться)
2. Заполните форму:
   - **Email** — ваш email (будет логином)
   - **Name** — ваше имя (можно псевдоним)
   - **Password** — пароль (запишите!)
3. Примите условия использования
4. Нажмите **"Register"**
5. Проверьте почту и подтвердите регистрацию

### Шаг 3: Войдите в систему

После подтверждения:
1. Вернитесь на главную страницу
2. Введите email и пароль в левой колонке
3. Нажмите **"Login"**

## Часть 2: Загрузка данных

### Откуда взять файл?

Для GEDmatch нужен файл с аутосомными данными. Это **НЕ** BAM-файл (он для Y-хромосомы).

**Источники аутосомных данных для участников AADNA:**

| Источник | Файл | Примечание |
|----------|------|------------|
| WGS (AADNA) | VCF или специальный экспорт | Спросите администратора |
| 23andMe | Скачивается из аккаунта | Формат txt |
| AncestryDNA | Скачивается из аккаунта | Формат txt |
| MyHeritage | Скачивается из аккаунта | Формат csv |
| FTDNA | Скачивается из аккаунта | Формат csv |

### Шаг 1: Подготовьте файл

Убедитесь, что у вас есть:
- Файл в формате txt, csv или zip
- Размер обычно 10-30 МБ
- Это должны быть «сырые данные» (raw data), не отчёт

### Шаг 2: Загрузите данные

1. После входа найдите раздел **"DNA Applications"** или **"Generic Uploads"**
2. Нажмите **"Generic DNA Uploads"** → **"Upload a DNA file"**
3. Заполните форму:

```
Name: Ваше имя (или псевдоним)
Optional: Email для уведомлений
Source: Выберите источник (23andMe, AncestryDNA, etc.)
         Если WGS — выберите "Other"
```

4. Нажмите **"Choose File"** и выберите ваш файл
5. Нажмите **"Upload"**

### Шаг 3: Дождитесь обработки

- Загрузка: 1-5 минут (зависит от интернета)
- Обработка: от 1 часа до 2 дней
- Вы получите email, когда данные будут готовы

### Шаг 4: Запишите ваш Kit Number

После обработки вы получите **Kit Number** — уникальный номер вашего набора данных.

Формат: буква + цифры, например: **A123456** или **M987654**

**Запишите его! Он понадобится для всех операций.**

## Часть 3: Поиск родственников (One-to-Many)

Это главная функция GEDmatch — поиск всех людей, с которыми у вас есть общая ДНК.

### Шаг 1: Запустите сравнение

1. Найдите раздел **"DNA Applications"**
2. Нажмите **"One-to-Many DNA Comparison"**
3. Введите ваш **Kit Number**
4. Оставьте настройки по умолчанию или измените:
   - **cM threshold**: минимум cM для показа (по умолчанию 7)
   - **Max results**: сколько результатов показывать
5. Нажмите **"Display Results"**

### Шаг 2: Понимание результатов

Вы увидите таблицу с колонками:

| Колонка | Значение |
|---------|----------|
| **Kit** | Номер набора совпадения |
| **Name** | Имя (или псевдоним) |
| **Email** | Контакт (если разрешён) |
| **Largest Seg** | Самый большой общий сегмент (cM) |
| **Total cM** | Общее количество cM |
| **Gen** | Примерное количество поколений до общего предка |
| **Overlap** | Количество общих SNP |

### Шаг 3: Интерпретация cM

| cM | Вероятное родство | Что делать |
|----|-------------------|------------|
| **200+** | Двоюродные или ближе | Обязательно пишите! |
| **100-200** | Троюродные | Стоит связаться |
| **50-100** | Четвероюродные | Для AADNA — ценные совпадения |
| **20-50** | 5-6 колено | Исследовать, если есть другие признаки |
| **7-20** | Очень дальние | Только при доп. подтверждениях |

### Шаг 4: Фильтрация результатов

Используйте браузерный поиск (Ctrl+F) для фильтрации по:
- Странам (Turkey, Jordan, Syria)
- Фамилиям
- Ключевым словам

## Часть 4: Детальное сравнение (One-to-One)

Когда вы нашли интересное совпадение, сравните детально.

### Шаг 1: Запустите сравнение

1. В разделе **"DNA Applications"**
2. Нажмите **"One-to-One DNA Comparison"**
3. Введите:
   - **Kit 1**: ваш номер
   - **Kit 2**: номер совпадения
4. Нажмите **"Submit"**

### Шаг 2: Понимание результатов

Вы увидите:

**Сводка:**
```
Largest segment = 45.2 cM
Total Half-Match segments (HIR) = 127.5 cM
Estimated number of generations to MRCA = 3.5
```

**Визуализация хромосом:**
Цветные полоски показывают, где именно совпадает ДНК:
- Каждая линия — хромосома (1-22)
- Цветные участки — совпадающие сегменты
- Длина участка = размер совпадения

### Шаг 3: Что смотреть?

1. **Largest segment** (крупнейший сегмент)
   - Чем больше — тем надёжнее родство
   - > 20 cM = вероятно реальный родственник
   - < 10 cM = может быть случайным

2. **Количество сегментов**
   - Много мелких сегментов = возможно, дальнее родство через несколько линий
   - Несколько крупных сегментов = более близкое родство

3. **Распределение по хромосомам**
   - Если совпадение только на одной хромосоме — родство по одной линии
   - Если на многих — возможно, родство по нескольким линиям

## Часть 5: Поиск общих совпадений (People who match both)

Эта функция ищет людей, которые совпадают и с вами, и с другим человеком. Помогает подтвердить родство и найти общую ветку.

### Как использовать

1. Раздел **"DNA Applications"**
2. **"People who match both kits"** или **"3D Chromosome Browser"**
3. Введите ваш kit и kit совпадения
4. Смотрите список общих совпадений

### Что это даёт?

Если вы и ваше совпадение оба совпадаете с третьим человеком — вероятно, вы все родственники через общего предка.

## Часть 6: Этнический калькулятор (Admixture)

GEDmatch предлагает несколько калькуляторов для оценки этнического происхождения.

### Шаг 1: Выбор калькулятора

1. Раздел **"Admixture (heritage)"**
2. Выберите калькулятор:
   - **MDLP** — хорош для Кавказа
   - **Dodecad** — популярный, подробный
   - **Eurogenes** — детализация европейских компонентов
   - **GedrosiaDNA** — для Ближнего Востока/Кавказа

### Шаг 2: Для кавказцев рекомендуется

**MDLP K23b** или **Dodecad K12b** — показывают:
- Кавказский компонент
- Ближневосточный компонент
- Европейские компоненты

### Шаг 3: Интерпретация

Результаты показывают проценты компонентов:
```
Пример для адыга:
- Caucasian: 65%
- West Asian: 20%
- European: 12%
- Other: 3%
```

**Важно:** Калькуляторы дают примерную картину. Не воспринимайте проценты буквально.

## Часть 7: Настройки приватности

### Как управлять видимостью

1. Войдите в аккаунт
2. Найдите **"Profile"** или **"Settings"**
3. Для каждого kit можно настроить:
   - **Public** — все видят ваш email и могут сравнивать
   - **Public but opt-out email** — сравнивают, но email скрыт
   - **Private** — только вы можете использовать данные
   - **Research** — участие в исследованиях

### Рекомендации для AADNA

- Установите **Public** для максимального поиска родственников
- Используйте отдельный email, если переживаете о спаме
- Укажите в имени что-то идентифицирующее (Circassian, Adyghe, Abkhaz)

## Часть 8: Стратегия поиска для AADNA

### Цель 1: Найти родственников в диаспоре

1. Запустите **One-to-Many**
2. Ищите совпадения с:
   - Turkey, TR
   - Jordan, JO
   - Syria, SY
   - Israel, IL
3. Фильтруйте по cM > 50
4. Пишите найденным совпадениям

### Цель 2: Подтвердить Y-ДНК родство

Если на YFull вы нашли генетического соседа:
1. Попросите его загрузить аутосомы на GEDmatch
2. Сделайте **One-to-One** сравнение
3. Если есть аутосомное совпадение — родство подтверждено дважды

### Цель 3: Построить общую картину

1. Выпишите все совпадения > 50 cM
2. Для каждого сделайте **One-to-One**
3. Проверьте **People who match both**
4. Группируйте совпадения по вероятным линиям

## Часть 10: Частые вопросы

### «Мало совпадений. Это нормально?»

Да, если вы из популяции, которая мало представлена в базе. Адыгов и абхазов пока немного в GEDmatch. Решение: привлекайте родственников к тестированию.

### «Совпадение не отвечает. Что делать?»

- Подождите 2-3 недели
- Попробуйте написать ещё раз
- Человек мог забросить аккаунт
- Ничего личного — продолжайте искать

### «Как узнать, по какой линии родство?»

1. Используйте **Chromosome Browser** для визуализации
2. Попросите совпадение тестировать родителей
3. Сравните с другими совпадениями через **People who match both**

### «Что такое Genesis?»

**GEDmatch Genesis** — новая версия платформы. Там те же функции, но обновлённый интерфейс. Постепенно Genesis заменит старую версию.

### «Нужен ли платный аккаунт?»

Базовые функции (One-to-Many, One-to-One, Admixture) — бесплатны. Платный аккаунт ($10/месяц) даёт:
- Расширенные инструменты (Tier 1)
- Больше калькуляторов
- Приоритетную обработку

Для начала бесплатного достаточно.

### «Безопасно ли загружать ДНК?»

- GEDmatch использует данные для генеалогии
- Были случаи использования полицией (с 2019 — по согласию)
- Вы можете удалить данные в любой момент
- Решение за вами — взвесьте риски и пользу

## Часть 11: Продвинутые инструменты

### Segment Search

Ищет совпадения на конкретном участке хромосомы:
1. Укажите хромосому, начало и конец
2. Найдёте всех, кто совпадает именно там
3. Полезно для триангуляции

### Are your parents related?

Проверяет, не были ли ваши родители родственниками (инбридинг):
- Если есть длинные участки гомозиготности — да
- Характерно для изолированных популяций
- Не означает близкое родство — может быть дальнее

### X-DNA comparison

Сравнивает X-хромосому отдельно:
- X передаётся особым образом
- Может уточнить, через какую линию родство

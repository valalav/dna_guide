### 6.3. Гипотеза «Восточной Экспансии»

> [!NOTE]
> **Y32924** доминирует у кабардинцев и присутствует у тюрков (балкарцы, карачаевцы).
> Это может указывать на:
> 1. **Раннее разделение** восточных и западных адыгов (~500 BCE)
> 2. **Контакты с Кобанской культурой** (прото-нахские связи?)
> 3. **Интеграцию тюркского элемента** через браки (аланско-булгарский период)

## 9. Историческая Интерпретация

### 9.1. Хронологическая Привязка

| Период | Дата | Событие | Генетический маркер |
|--------|------|---------|-------------------|
| **Коллапс Майкопа** | ~2200 BCE | Начало дольменной культуры | Формирование FT9681 |
| **Расцвет дольменов** | ~1500 BCE | Дифференциация племён | Z44239 → Y32606/Y32924 |
| **Кобанская культура** | ~1100-500 BCE | Железный Век | Y32924 экспансия |
| **Сарматский период** | 200 BCE – 400 CE | Степное влияние | FT73578 → поздние клады |
| **Формирование Кабарды** | 500-1500 CE | Этногенез адыгов | Y219946 bottleneck |

### 9.2. Почему Кабардинцы Доминируют?

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                      ГИПОТЕЗА ВОСТОЧНОЙ ЭКСПАНСИИ                           │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  1. FT9681 формируется в ЦЕНТРАЛЬНОМ Кавказе (~1900 BCE)                   │
│     │                                                                       │
│  2. Бифуркация Z44239 → Y32924 создаёт «восточный клан» (~1500 BCE)        │
│     │                                                                       │

│     │                                                                       │
│  4. Founder effect: Y32924 становится доминирующей линией кабардинцев      │
│     │                                                                       │
```
